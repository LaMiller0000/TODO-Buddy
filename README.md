# Syntax Buddy
    A todo app made to motivate students with your studies!

    Our app brings you:
        - a fun little friend
        - real life consequences
        - cute rewards!

    Features:
        - Todo list
            - Task dependancies
            - Prioritzation
            - Task Web
        - Buddy
            - Customizable friend
            - Health
